Añadiendo flexbox a nuestros div

En el siguente ejemplo tenemos una lista de elementos 

                    <section>
    
                        <div>1</div>
    
                        <div>2</div>
    
                        <div>3</div>
    
                        <div>4</div>
    
                        <div>5</div>
    
                    </section>
            

            .container{
                display: flex;
                flex-direction: row;
                background-color: brown;
                padding: 10px;
                border: 2px solid black;
            }
            
            .container div {
                background-color: burlywood;
                width: 100px;
                margin: 5px;
                text-align: center;
                font-size: 20px;
                border: 2px solid black;
            }
